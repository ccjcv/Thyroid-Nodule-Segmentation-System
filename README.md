# Thyroid-Nodule-Segmentation-System
基于QT，mysql与Transformer的甲状腺图像分割系统。（软著已登记）

项目描述：使用QT进行开发，配置基于mysql的用户登录系统以及子线程进行客户端与服务端之间的socker通信，子线程传输图像文件，后端使用python图像分割神经网络进行分割，分割结果于界面显示。
